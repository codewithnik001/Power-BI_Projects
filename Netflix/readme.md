# Netflix Content Analysis Dashboard

## 📊 Project Overview

This Power BI project provides a comprehensive visualization of Netflix's content library. The dashboard is designed to analyze the distribution of Movies vs. TV Shows, geographical content availability, and historical release trends to identify patterns in Netflix's global streaming strategy.

## 🛠️ Data Architecture

The project utilizes a structured Data Model optimized for Power BI's analytical engine:

* **Core Model:** Built on the `DataModel` schema containing content metadata.
* **Theme:** Uses the modern `CY24SU10` (October 2024) base theme for high-contrast, accessible reporting.
* **Static Assets:** Includes high-fidelity vector branding (Netflix Logo Assets) for professional UI/UX design.

## 🚀 Key Features

* **Content Split Analysis:** Comparative metrics between "Movies" and "TV Shows" to see where Netflix is investing its budget.
* **Geographical Mapping:** Visualizes content production and availability by country using Power BI’s map integration.
* **Temporal Trends:** Timeline analysis of content releases from early years to the current 2024–2026 data points.
* **Category Deep-Dives:** Analysis of ratings, genres, and maturity levels across different regions.

## 📂 File Structure

| File Component | Description |
| --- | --- |
| `DataModel` | The underlying compressed binary data containing Netflix datasets. |
| `Report/Layout` | Defines the visual arrangement and canvas settings of the dashboard. |
| `BaseThemes/` | Custom JSON styling for consistent Netflix-branded color palettes. |
| `RegisteredResources/` | Contains the Netflix vector logos and UI iconography. |

## 📈 Insights Target

This dashboard aims to answer:

1. Which countries are the top contributors to the Netflix library?
2. How has the ratio of TV Shows changed relative to Movies over the last decade?
3. What is the most common maturity rating for content added in the last 24 months?
